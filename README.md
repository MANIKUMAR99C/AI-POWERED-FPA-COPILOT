# AI-Powered FP&A Copilot

This project demonstrates how core FP&A workflows can be automated using Python
and AI-assisted logic.

## What this project does
- Budget vs Actual variance analysis
- Variance percentage calculation
- Overspend identification and prioritization
- Executive-style commentary generation
- Automated monthly finance summary

## Why I built this
I built this project to show how finance teams can reduce manual reporting
and focus more on decision-ready insights using automation and AI.

## Tools used
- Python (pandas)
- Google Colab
- AI-assisted debugging (Gemini)

## Data
The dataset is a mock FP&A dataset designed to reflect real enterprise
cost centers and spend categories.
